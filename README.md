# MiniShop REST API

![CI Pipeline](https://github.com/Djacobocolq/MiniShop/actions/workflows/ci.yml/badge.svg)

Proyecto desarrollado con Spring Boot, Maven, JPA, H2 y GitHub Actions.

## Funcionalidades

- Crear productos
- Listar productos
- Buscar productos por ID
- Validaciones de negocio
- Pruebas de integración
- Pipeline CI/CD con GitHub Actions
